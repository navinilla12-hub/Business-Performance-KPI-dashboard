# Business-Performance-KPI-dashboard

# Overview

This project presents an end-to-end Business Intelligence dashboard built to analyze business performance across sales, profitability, and customer activity. The goal was to transform raw transactional data into actionable insights that support data-driven decision-making.

The dashboard enables stakeholders to monitor key performance indicators (KPIs), identify trends, and uncover business opportunities.

# Objectives
Track key business KPIs such as revenue, profit, and order volume
Analyze sales trends over time
Identify top-performing products and regions
Enable interactive filtering for dynamic analysis
Support business decision-making through visual insights

# Tools & Technologies
Power BI (Web)
SQL (for data understanding)
Microsoft Excel (data preprocessing)

# Dataset

Dataset used: Superstore Sales Dataset (public dataset)

# Key fields:

Order Date
Sales
Profit
Region
Category
Product Name
